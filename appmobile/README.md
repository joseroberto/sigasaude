# SIGA SAÚDE

Projeto feito utilizando o framework Flutter (Linguagem Dart)

## Iniciando

O pré-requisito para instalação é:

- Android Studio
- Flutter app
- Android SDK
- XCode (para desenvolvimento da parte IOS)


## Desenvolvimento

Para iniciar/executar a aplicação utlize o comando `flutter run`
